# CursoHerramientasBasicasFalcone
Este repositorio esta creado para presentar mi trabajo final del curso Herramientas básicas para el análisis de datos de UTN

## Descripción
Este proyecto integra los conocimientos adquiridos a lo largo del curso de análisis de datos, abordando el proceso completo desde la comprensión de los datos hasta su visualización y comunicación.

Se trabajó con un dataset de productividad estudiantil para analizar la relación entre hábitos (como horas de sueño) y el rendimiento académico, aplicando técnicas de análisis exploratorio, limpieza de datos, visualización y construcción de dashboards.

## Objetivo
Transformar datos en información útil que permita identificar patrones y comunicar resultados de manera clara mediante herramientas analíticas y visuales.

## Dataset
Tipo de datos: estructurados (tabla)
Variables principales:
Horas de sueño
Nota final
Variables de hábitos

## Metodología

El proyecto se desarrolló siguiendo un flujo de trabajo típico de análisis de datos:

### Exploración de datos
Análisis de estructura 
Estadísticas descriptivas 
Identificación de valores nulos y duplicados

### Limpieza de datos
Tratamiento de valores faltantes (mediana/moda)
Eliminación de inconsistencias
Corrección de tipos de datos

### Transformación
Creación de variables derivadas
Agrupación de datos
Cálculo de métricas 

### Visualización en Python
Gráficos de dispersión → relación entre variables
Gráficos de promedios → tendencias
Análisis de distribución

### Dashboard en Power BI

Se construyó un dashboard para comunicar resultados de forma clara:

Tarjetas KPI:
Total de alumnos
Cantidad de aprobados (nota ≥ 60)
Gráficos comparativos:
Aprobados vs desaprobados

### Resultados principales
Se identificó una leve relación positiva entre horas de sueño y rendimiento académico
A mayor cantidad de horas de descanso, el promedio de notas tiende a mejorar
No se observó una relación fuerte o determinante entre las variables
Fue necesario aplicar transformaciones para visualizar mejor los patrones
